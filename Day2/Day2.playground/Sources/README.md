## 문자열을 결합하는 법

*String Interpolation* 예시
```
let name = "Taylor"
let age = 26
let message = "Hello, my name is \(name) and I'm \(age) years old."
print(message)
```

문자열 안에 정수, 소수 등을 다른 변형 없이 사용가능하다.
또한 `string interpolation` 구문 내부에 연산이 가능하다.
